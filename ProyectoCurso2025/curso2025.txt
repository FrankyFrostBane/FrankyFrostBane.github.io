    programacion web
    La especialización en programación web, brinda al programador conocimiento y habilidades de aplicación 
    especifica: maquetar interfaces de acuardo con el diseño dado.
    Programar interactividad, mantener, optimizar, actualizar sitios web.
    Programar procedimientos almacenados y transacciones en las bases de datos.
    Desarrollar sitios web que utilicen base de datos.
    Producir documentación técnica de proceso de productos desarrollados.

    diseño web
    La especialización en diseño web brinda al diseñador habilidades para crear y diseñar sitios web  que 
    cumplan con los pedidos del cliente.
    Planficar la estructura del sitio, diseñar la experiencia de usuario, crear objetos gráficos para la 
    visualización en pantalla, optimizar sitios web. 
    Instalar y configurar el gestor de contenidos, Cuidar la seguridad del software de diseño web.
    Configuración de sitios y archivos de proyectos. Creación de un diseño de página, Html5, marcadores, 
    tablas, inserción de imagen, animaciones y videos, inserción de texto creación de vínculos, 
    visualización de la página, creación de hoja de estilo css, adjuntar hoja de estilos, diseños por capas. etc.
